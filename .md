State (useState) Updates automatically, causes re-render

Re-render = React updates the UI with new values

Ref (useRef) Store value, doesn't update automatically, doesn't cause re-render

State (useState) → Update causes re-render (UI updates)
Ref (useRef) → Update does NOT cause re-render (UI doesn't update)

That's why useRef is used for tracking old values without affecting the UI! 👍

When a user submits an HTML form, the data comes in URL-encoded format:
name=John&email=john@gmail.com&age=25
This middleware converts it to a JavaScript object:
```
{
  name: "John",
  email: "john@gmail.com",
  age: "25"
}
```

Then you access it in routes via req.body:
```
app.post('/register', (req, res) => {
  console.log(req.body.name)   // "John"
  console.log(req.body.email)  // "john@gmail.com"
})
```
